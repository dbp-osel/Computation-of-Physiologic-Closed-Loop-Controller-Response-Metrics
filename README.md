General Information

This software package provides tools to compute physiologic closed-loop controller (PCLC) performance metrics from time-series data of a controlled physiologic variable as described in [1] Chalumuri et al. Annals of Biomedical Engineering, 54, pp. 857–868 (2026). https://doi.org/10.1007/s10439-025-03929-2. The package is available in both MATLAB and Python.
